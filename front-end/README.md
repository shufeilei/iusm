# IUSM Front-End Code
### For deploying use:
- `index.html`
- `styles/``
- `dist/``
- `images/``

### For development:
1. brew install node
1. npm install -g yarn
1. yarn install
1. yarn build
1. sass css/style.scss styles/app.css
1. open index.html

### For editing upcoming events:
- Upcoming events: edit the upcomingEventsData array in `src/constants.js`
